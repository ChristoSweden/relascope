# Testa Digital Relaskop — så här gör du

Tack för att du testar! Det här tar ca 20–30 minuter i skogen och kräver
ingen förkunskap utöver att du har använt ett vanligt relaskop.

**Appen:** <https://christosweden.github.io/relascope/>
**Feedback:** <christo@beetlesense.com>

## 1. Installera (30 sekunder, hemma med täckning)

1. Öppna länken ovan i mobilens webbläsare.
2. Lägg till på hemskärmen: **iPhone** — dela-knappen → "Lägg till på
   hemskärmen". **Android** — menyn (⋮) → "Installera app".
3. Öppna appen en gång. Därefter fungerar den helt offline.

## 2. Kalibrera (5 minuter, en gång per telefon)

Appen behöver veta kamerans synfält. Följ guiden under **Kalibrera**: ställ
ett föremål med känd bredd (t.ex. ett A4-papper, 21 cm) på känt avstånd och
markera kanterna. Kör gärna **Verifiera** efteråt — appen säger själv om
kalibreringen håller måttet.

## 3. Mät — appen mot ditt relaskop

1. Välj 3–5 provytor i ett bestånd du känner. Markera punkten (pinne/spray).
2. **Mät med ditt vanliga relaskop först** (faktor 2). Anteckna grundytan.
3. Mät samma punkt med appen: nytt bestånd → **Nytt svep** → gå runt 360°
   och peka ut varje stam som TRÄFF / GRÄNSTRÄD / EJ TRÄFF.
4. Stå exakt på samma punkt för båda mätningarna och räkna samma varv —
   skillnaden vi vill mäta är verktyget, inte ståndpunkten.

## 4. Skicka in

Mejla till **christo@beetlesense.com**:

- Grundytan per provyta: ditt relaskop vs appen (eller CSV-exporten från
  appen + dina anteckningar).
- Telefonmodell.
- Det viktigaste: **kändes siffrorna trovärdiga?** Var gränsträden svårare
  eller lättare än med ett vanligt relaskop?
- Allt som var krångligt, otydligt eller irriterande — småsaker räknas.

Som tumregel: hamnar appen inom ±1 träff per provyta mot ditt relaskop är
det godkänt. Större avvikelser vill vi verkligen höra om.
